# 💳 Credit Card Customer Analysis

## 📊 Project Overview

This project focuses on analyzing credit card customer data to uncover insights related to customer behavior, risk segmentation, and revenue generation. The analysis is performed using SQL and visualized through interactive Power BI dashboards.

---

## 🎯 Objectives

* Analyze customer demographics and distribution
* Identify high-risk and low-risk customer segments
* Understand transaction behavior and usage patterns
* Evaluate revenue and interest generation drivers
* Provide actionable business insights

---

## 🛠 Tools & Technologies

* **SQL** – Data querying and analysis
* **Power BI** – Data visualization and dashboard creation

---

## 📊 Dashboards

### 🔵 General Dashboard

* Customer count and key KPIs
* Card category distribution
* Customer demographics (education, gender, marital status)
* Transaction behavior analysis

---

### 🔴 Risk Segmentation Dashboard

* Customer segmentation (Low, Medium, High Risk)
* Risk distribution by card type and transaction method
* Personal loan analysis
* Risk-focused insights

---

### 🟢 Interest Earned Dashboard

* Interest contribution by risk segments
* Interest by card category and transaction type
* Spending category analysis
* Revenue-driving factors

---

## 🧠 Key Insights

* Low-risk customers generate the majority of interest revenue
* Blue card holders dominate both customer base and revenue contribution
* Swipe transactions contribute the highest transaction volume
* High-risk customers form a small but critical segment requiring monitoring
* Essential spending categories (bills, fuel, groceries) drive consistent revenue
